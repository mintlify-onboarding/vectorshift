---
title: "Text"
description: "Text node allows you to write text and input it to another node."
---

Notes:

1. You can use brackets to variables `(view "{{placeholder}}")` below. Whatever ID is enclosed in the brackets will appear on the left-hand side of the block. Information/data connected to the node will "replace" the bracket in the output of the text block

2. Common use cases for text nodes  
   1. Use to compose instructions for the "system" input for LLMs  
   2. Use to compose prompts for the "prompt" input for LLMs  
   3. Include a link to a website to be connected to a URL data loader
